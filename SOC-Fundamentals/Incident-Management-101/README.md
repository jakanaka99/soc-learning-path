## ğŸ§  Key Learnings
- **Incident:** A confirmed security problem.  
- **Event:** Any recorded system activity (not always bad).  
- **Alert:** A warning from a tool about possible danger.  
- **False Positive:** An alert that looks bad but is actually safe.  
- **True Positive:** A real, confirmed threat.  
- **False Negative:** A real attack that the system missed.  
- **Playbook:** A predefined, step-by-step guide for handling specific types of security incidents.
